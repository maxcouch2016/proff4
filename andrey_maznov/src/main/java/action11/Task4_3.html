Задача 4. Реализовать 3 html файла, в которых будет 3 частично перекрывающие друг друга квадрата.

На странице 1 первый квадрат поверх остальных.

На странице 2 второй квадрат поверх остальных.

На странице 3 третий квадрат поверх остальных.

Размеры квадратов одиннаковые и одиннаковая рамочка.

Каждый квадрат является ссылкой на соответсвующую страницу 1, 2 и 3.

Таким образом нажимая на кнопку 1-поверх становится квадрат 1.

Таким образом нажимая на кнопку 2-поверх становится квадрат 2.

Таким образом нажимая на кнопку 3-поверх становится квадрат 3.
